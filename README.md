## Description

This project describes the utilization of graphql in combination with postgresSQL

## Installation

```bash
$ npm install
or
$ yarn install
```

## Running the app

```bash
# development
$ npm run start
or
$ yarn start

# watch mode
$ npm run start:dev
or
$ yarn start:dev

# production mode
$ npm run start:prod
or
$ yarn start:prod
```
